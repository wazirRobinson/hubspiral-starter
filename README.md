# Agency Starter Template v1

Frozen on: 2026-01-15

## Requirements

- Node.js (LTS recommended)
- npm

## Setup

1. Install dependencies
   npm install

2. Create .env
   Copy .env.example to .env and fill in values

3. Customize branding/content
   Edit config/siteConfig.js

4. Run locally
   npm run dev

## Key folders

- config/ (client branding/content)
- routes/ (Express routes)
- services/ (Stripe + email services)
- views/ (Handlebars templates)
- public/ (static assets)
